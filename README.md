# ? JavaScript 스프레드시트 웹 애플리케이션  

이 프로젝트는 JavaScript 학습 여정을 통해 얻은 지식들을 바탕으로 완성한 **웹 기반 스프레드시트 애플리케이션**입니다.

---

## ? 주요 기능

- ### ? 직관적인 그리드 UI  
  깔끔하고 반응성 높은 스프레드시트 그리드를 통해 데이터를 손쉽게 입력하고 수정할 수 있습니다.

- ### ? 스마트 셀 하이라이트
  - 선택한 셀의 **행 헤더(왼쪽)** 및 **열 헤더(상단)**가 **산뜻한 하늘색**으로 하이라이트되어 현재 위치를 쉽게 파악할 수 있습니다.
  - 선택된 셀의 주소(예: `A1`)는 화면 하단 `Cell:` 상태 표시줄에 **실시간 표시**됩니다.

- ### ? Excel 내보내기 (Export)
  - "Export Spreadsheet" 버튼 클릭 한 번으로 현재 시트 데이터를 `.xlsx` 파일로 저장할 수 있습니다.
  - Google 스프레드시트, MS Excel 등 다양한 프로그램과 **완벽하게 호환**됩니다.

- ### ? Excel 가져오기 (Import)
  - `.xlsx` 또는 `.xls` 파일을 앱에 불러와 데이터로 바로 적용할 수 있습니다.
  - Google 스프레드시트에서 작성한 파일도 문제없이 불러올 수 있습니다.

## ? 사용 기술 스택

- **HTML5** ? 웹 페이지 구조 설계
- **CSS3** ? 반응형 인터페이스 및 스타일링
- **JavaScript (ES6+)** ? 사용자 인터랙션 및 로직 처리
- **[SheetJS (js-xlsx)](https://sheetjs.com/)** ? Excel 파일 입출력 및 파싱

---

## ? 시작하는 방법

1. **앱 실행**  
   프로젝트 폴더에서 `index.html` 파일을 브라우저로 열어 실행합니다.  
   (※ VS Code 사용자라면 Live Server 확장 기능 사용을 추천합니다.)

2. **데이터 입력 및 편집**  
   셀을 클릭하여 데이터를 자유롭게 입력하거나 수정하세요.

3. **Excel로 내보내기**  
   `Export Spreadsheet` 버튼을 클릭하면 `.xlsx` 파일이 다운로드됩니다.

4. **Excel에서 가져오기**  
   `파일 선택` 버튼으로 기존 Excel 파일을 불러오고 데이터를 확인하세요.
